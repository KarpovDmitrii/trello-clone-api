
# Тестовое задание: Backend аналога Trello.

## Стек:
- Framework: NestJS
- Database: PostgreSQL + Kysely
- Auth: JWT
- Documentation: Swagger

## Запуск проекта

1. Установка зависимостей:
   ```bash
   npm install
   ```
2. Настройка окружения:
   В корне проекта создать файл .env по примеру .env.example

4. Миграции:
    ```bash
    npm run db:migration:run
   ```

5. Запуск:
   ```bash
   npm run start:dev
   ```

После запуска Swagger доступен по адресу: 'http://localhost:3000/docs'

## Права доступа
#### Пользователи:
- Регистрация и логин доступны всем.
- Просмотр профиля (GET /users/:id) доступен любому авторизованному пользователю.
- Редактирование профиля (PATCH /users/me) разрешено только самому владельцу аккаунта (ID извлекается из JWT).
#### Колонки:
- Создавать колонки может любой авторизованный пользователь.
- Просматривать колонки может любой залогиненный юзер.
- Редактировать и удалять колонку может только её создатель.
#### Карточки:
- Создавать карточки может только владелец колонки, в которой они создаются.
- Редактировать и удалять карточку может также только владелец родительской колонки.
- Просмотр карточек доступен всем авторизованным пользователям .
#### Комментарии:
- Оставлять комментарии к карточкам может любой авторизованный пользователь.
- Редактировать и удалять комментарий может только его автор.

При удалении родительского обьекта, доступ к дочернему обьекту блокируется
